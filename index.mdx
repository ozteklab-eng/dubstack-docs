## Product surfaces

- Dashboard: manage sites, editor workflows, team access, billing, and settings.
- Docs Shell: render clean documentation sites from Git-backed MDX and `dubstack.json`.
- Theme Studio: customize Enterprise docs experiences with Canvas Layout.
- Public wiki previews: preview public repositories before claim/adopt.
- Control Plane: internal DubStack operations and lifecycle management.
