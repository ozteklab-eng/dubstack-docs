## What ships in the starter

The default starter includes three API sections:

- **Beat patterns** for creating, playing, exporting, and controlling beats
- **Drum kits** for inspecting kits, sounds, and custom mixes
- **Sessions** for checking and clearing the active session

## How to customize this

1. Replace `api-reference/openapi.json` with your own OpenAPI spec
2. Use the dashboard to generate snippets when you need them
3. Publish only the endpoint pages you want visible in navigation

<Callout type="success">
DubStack keeps the uploaded spec as the source of truth. The playground reads from the spec, while endpoint pages stay explicit and controllable.
</Callout>
