Returns a specific pre-seeded demo beat pattern by ID.

> `GET /beats/{id}`

### Parameters
- `id` (path, string, required) · Beat pattern ID · example: `dembow`

### Responses
- `200` Beat pattern found.
- `404` Beat pattern not found.
