Returns a specific drum kit with its full sound definitions and control parameters for tuning and decay adjustments.

> `GET /kits/{id}`

### Parameters
- `id` (path, string, required) · Kit ID · example: `tr-808`

### Responses
- `200` Drum kit found with full details and controls.
- `404` Kit not found.
