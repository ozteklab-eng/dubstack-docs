Returns all available sounds/instruments in a specific drum kit with character descriptions and sample hints.

> `GET /kits/{id}/sounds`

### Parameters
- `id` (path, string, required) · Kit ID · example: `acoustic`

### Responses
- `200` Kit sounds listed.
- `404` Kit not found.
