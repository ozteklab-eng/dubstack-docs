Stops the current playback session for a beat pattern and returns session statistics including total play time and loop count.

> `POST /beats/{id}/stop`

### Parameters
- `id` (path, string, required) · Beat pattern ID to stop · example: `boom-bap`

### Responses
- `200` Playback stopped successfully. Returns session statistics.
- `404` No active playback session found for this beat.
