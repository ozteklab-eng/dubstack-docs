Explicitly clears all session data including custom beats, mixes, and playback state. This is equivalent to logging out - all custom-created resources are permanently deleted. Sessions also auto-expire after the TTL (default 1 hour of inactivity).

> `DELETE /sessions/me`

### Responses
- `200` Session cleared. All custom data deleted.
